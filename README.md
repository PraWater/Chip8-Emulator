A Chip 8 Emulator written in C++ using SDL2.

### Requirements
- SDL2 Library

### How to run
Drag and drop a rom on the executable or run using the terminal.

```
./Chip8_Emulator {path_to_rom}
```

### Building
```
mkdir build
cd build
cmake ..
make
```
